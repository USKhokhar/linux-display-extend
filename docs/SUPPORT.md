# Support

## Where To Ask For Help

For general usage questions, installation help, and troubleshooting:

- use the GitHub issue templates when the problem looks like a bug or install problem
- use GitHub Discussions if that space is enabled for broader questions and ideas

## Before Opening An Issue

Please gather the following first:

- Linux distribution and version
- whether the session is X11 or Wayland
- output of `display-extend doctor`
- output of `display-extend status`
- relevant output from `display-extend logs`
- `xrandr --query` when the issue is display-related

## What This Project Supports Today

- Linux hosts
- X11 sessions
- Android clients using a VNC viewer

## What This Project Does Not Yet Support

- Wayland sessions
- encrypted transport built into the tool itself
- multi-device display layouts
