# Code of Conduct

## Our Commitment

We want Linux Display Extend to be an open, respectful, and collaborative project for users, contributors, and maintainers.

Participation in this project means agreeing to help create an environment that is:

- welcoming
- constructive
- respectful of differing backgrounds and experience levels
- focused on the health of the project and community

## Expected Behavior

Examples of behavior that help this project:

- giving actionable, respectful feedback
- assuming good intent while staying honest about technical risks
- helping new contributors understand project constraints
- keeping discussions focused on the code, docs, and product

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, intimidation, or discrimination
- personal attacks or insulting language
- bad-faith trolling or deliberately derailing technical discussion
- publishing someone else’s private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct. They may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that do not align with these standards.

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers through the contact channels documented in the repository or project profile.

## Scope

This code of conduct applies within project spaces and in public spaces when an individual is representing the project.

## Attribution

This document is informed by common open source community standards, including the Contributor Covenant.
